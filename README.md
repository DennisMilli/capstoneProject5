### Description
A dynamic Full-Stack Football Fan Analysis platform. Features live score, matches, table and goal chart, match analysis and news mini blog, real-time data flow, user interactions, article management, and a persistent PostgreSQL-backed commenting system.

### Tech Stack

Node.js

Express.js

PostgreSQL

EJS

Vanilla CSS

REST APIs

### Features
Live Scores, matches and table of different seasons of football

Real-time matchroom discussions

Mini blog for latest news and fan analysis of matches

Article management system

PostgreSQL-backed commenting system

CRUD operations for user-generated content
